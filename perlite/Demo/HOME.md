# Welcome to Perlite-Auth

#tag-test

**Perlite-Auth** is a fork of [secure-77/Perlite](https://github.com/secure-77/Perlite) — a web-based markdown viewer for your [Obsidian](https://obsidian.md/) vault.

## What's new in Perlite-Auth

- **Secure login** — the vault sits behind a username/password login screen, with the password stored as a bcrypt hash rather than plain text.
- **Bases support** — Obsidian's Bases views are rendered as read-only tables.
- **Better image display** — image embeds render correctly, including ones inserted via Obsidian's "Insert Attachment" or linked through table cells.

## Explore this demo vault

Browse the sidebar to see Perlite in action: markdown rendering, the graph view, LaTeX/Mermaid support, tags, and more.
